## <a name="create-a-virtual-machine-in-eclipse"></a><span data-ttu-id="40c56-105">在 Eclipse 中建立虛擬機器</span><span class="sxs-lookup"><span data-stu-id="40c56-105">Create a virtual machine in Eclipse</span></span>

<span data-ttu-id="40c56-106">若要使用 Azure Explorer 來建立虛擬機器，請執行下列作業︰</span><span class="sxs-lookup"><span data-stu-id="40c56-106">To create a virtual machine by using the Azure Explorer, do the following:</span></span>

1. <span data-ttu-id="40c56-107">使用 [Azure Toolkit for Eclipse 的登入指示](https://docs.microsoft.com/java/azure/eclipse/azure-toolkit-for-eclipse-sign-in-instructions) 來登入您的 Azure 帳戶。</span><span class="sxs-lookup"><span data-stu-id="40c56-107">Sign in to your Azure account by using the [Sign-in instructions for the Azure Toolkit for Eclipse](https://docs.microsoft.com/java/azure/eclipse/azure-toolkit-for-eclipse-sign-in-instructions).</span></span>

2. <span data-ttu-id="40c56-108">在 [Azure Explorer] 檢視中，展開 [Azure] 節點，以滑鼠右鍵按一下 [虛擬機器]，然後按一下 [建立 VM]。</span><span class="sxs-lookup"><span data-stu-id="40c56-108">In the **Azure Explorer** view, expand the **Azure** node, right-click **Virtual Machines**, and then click **Create VM**.</span></span>

   ![建立 VM 命令][CR01]  

   <span data-ttu-id="40c56-110">[建立新的虛擬機器] 精靈隨即開啟。</span><span class="sxs-lookup"><span data-stu-id="40c56-110">The **Create new Virtual Machine** wizard opens.</span></span>

3. <span data-ttu-id="40c56-111">在 [選擇訂用帳戶] 視窗中選取您的訂用帳戶，然後按 [下一步]。</span><span class="sxs-lookup"><span data-stu-id="40c56-111">In the **Choose a Subscription** window, select your subscription, and then click **Next**.</span></span>

   ![選擇訂用帳戶視窗][CR02]

4. <span data-ttu-id="40c56-113">在 [選取虛擬機器映像] 視窗中，輸入下列資訊︰</span><span class="sxs-lookup"><span data-stu-id="40c56-113">In the **Select a Virtual Machine Image** window, enter the following information:</span></span>

   * <span data-ttu-id="40c56-114">**位置**︰指定要建立虛擬機器的位置 (例如「美國西部」)。</span><span class="sxs-lookup"><span data-stu-id="40c56-114">**Location**: Specifies where your virtual machine will be created (for example, *West US*).</span></span>

   * <span data-ttu-id="40c56-115">**發行者**︰指定是哪個發行者建立您將用於建立虛擬機器的映像 (例如「Microsoft」)。</span><span class="sxs-lookup"><span data-stu-id="40c56-115">**Publisher**: Specifies the publisher that created the image you'll use to create your virtual machine (for example, *Microsoft*).</span></span>

   * <span data-ttu-id="40c56-116">**供應項目**︰從選取的發行者中指定要使用的虛擬機器供應項目 (例如「JDK」)。</span><span class="sxs-lookup"><span data-stu-id="40c56-116">**Offer**: Specifies the virtual machine offering to use from the selected publisher (for example, *JDK*).</span></span>

   * <span data-ttu-id="40c56-117">**SKU**︰從選取的供應項目中指定要使用的 Stockkeeping 單元 (SKU) (例如「JDK_8」)。</span><span class="sxs-lookup"><span data-stu-id="40c56-117">**Sku**: Specifies the stockkeeping unit (SKU) to use from the selected offering (for example, *JDK_8*).</span></span>

   * <span data-ttu-id="40c56-118">**版本 #**︰指定所選 SKU 要使用的版本。</span><span class="sxs-lookup"><span data-stu-id="40c56-118">**Version #**: Specifies which version of the selected SKU to use.</span></span>

   ![選取虛擬機器映像視窗][CR03]

5. <span data-ttu-id="40c56-120">按 [下一步] 。</span><span class="sxs-lookup"><span data-stu-id="40c56-120">Click **Next**.</span></span>

6. <span data-ttu-id="40c56-121">在 [虛擬機器基本設定] 視窗中，輸入下列資訊︰</span><span class="sxs-lookup"><span data-stu-id="40c56-121">In the **Virtual Machine Basic Settings** window, enter the following information:</span></span>

   * <span data-ttu-id="40c56-122">**虛擬機器名稱**：指定您新虛擬機器的名稱，其必須以字母開頭，且只能包含字母、數字及連字號。</span><span class="sxs-lookup"><span data-stu-id="40c56-122">**Virtual Machine Name**: Specifies the name for your new virtual machine, which must start with a letter and contain only letters, numbers, and hyphens.</span></span>

   * <span data-ttu-id="40c56-123">**大小**︰指定要配置給虛擬機器的核心和記憶體數目。</span><span class="sxs-lookup"><span data-stu-id="40c56-123">**Size**: Specifies the number of cores and memory to allocate for your virtual machine.</span></span>

   * <span data-ttu-id="40c56-124">**使用者名稱**︰指定要管理您的虛擬機器所建立的系統管理員帳戶。</span><span class="sxs-lookup"><span data-stu-id="40c56-124">**User name**: Specifies the administrator account to create for managing your virtual machine.</span></span>

   * <span data-ttu-id="40c56-125">**密碼**和**確認**︰指定您系統管理員帳戶的密碼。</span><span class="sxs-lookup"><span data-stu-id="40c56-125">**Password** and **Confirm**: Specifies the password for your administrator account.</span></span>

   ![虛擬機器基本設定視窗][CR04]

7. <span data-ttu-id="40c56-127">按 [下一步] 。</span><span class="sxs-lookup"><span data-stu-id="40c56-127">Click **Next**.</span></span>

8. <span data-ttu-id="40c56-128">在 [建立新的儲存體帳戶] 視窗中輸入下列資訊：</span><span class="sxs-lookup"><span data-stu-id="40c56-128">In the **Create New Storage Account** window, enter the following information:</span></span>

   * <span data-ttu-id="40c56-129">**資源群組**︰指定虛擬機器的資源群組。</span><span class="sxs-lookup"><span data-stu-id="40c56-129">**Resource Group**: Specifies the resource group for your virtual machine.</span></span> <span data-ttu-id="40c56-130">選取下列其中一個選項：</span><span class="sxs-lookup"><span data-stu-id="40c56-130">Select one of the following options:</span></span>
     * <span data-ttu-id="40c56-131">**新建**：指定您想要建立新的資源群組。</span><span class="sxs-lookup"><span data-stu-id="40c56-131">**Create new**: Specifies that you want to create a new resource group.</span></span>
     * <span data-ttu-id="40c56-132">**使用現有項目**︰指定您要選取已與您 Azure 帳戶相關聯的資源群組。</span><span class="sxs-lookup"><span data-stu-id="40c56-132">**Use existing**: Specifies that you want to select a resource group that is already associated with your Azure account.</span></span>

       ![新建儲存體帳戶對話方塊][CR05]

   * <span data-ttu-id="40c56-134">**儲存體帳戶**︰指定要用來儲存虛擬機器的儲存體帳戶。</span><span class="sxs-lookup"><span data-stu-id="40c56-134">**Storage account**: Specifies the storage account to use for storing your virtual machine.</span></span> <span data-ttu-id="40c56-135">您可以使用現有的儲存體帳戶或建立新帳戶。</span><span class="sxs-lookup"><span data-stu-id="40c56-135">You can use an existing storage account or create a new account.</span></span>

   * <span data-ttu-id="40c56-136">**虛擬網路**和**子網路**︰指定虛擬機器所要連線的虛擬網路和子網路。</span><span class="sxs-lookup"><span data-stu-id="40c56-136">**Virtual Network** and **Subnet**: Specifies the virtual network and subnet that your virtual machine will connect to.</span></span> <span data-ttu-id="40c56-137">您可以使用現有的網路和子網路，也可以建立新的網路和子網路。</span><span class="sxs-lookup"><span data-stu-id="40c56-137">You can use an existing network and subnet, or you can create a new network and subnet.</span></span> <span data-ttu-id="40c56-138">如果您選取 [新建]，畫面上會顯示下列對話方塊︰</span><span class="sxs-lookup"><span data-stu-id="40c56-138">If you select **Create new**, the following dialog box is displayed:</span></span>

      ![新建虛擬網路對話方塊][CR06]

9. <span data-ttu-id="40c56-140">在 [相關聯的資源] 視窗中，輸入下列資訊：</span><span class="sxs-lookup"><span data-stu-id="40c56-140">In the **Associated Resources** window, enter the following information:</span></span>

   * <span data-ttu-id="40c56-141">**公用 IP 位址**︰指定虛擬機器的對外 IP 位址。</span><span class="sxs-lookup"><span data-stu-id="40c56-141">**Public IP address**: Specifies an external-facing IP address for your virtual machine.</span></span> <span data-ttu-id="40c56-142">您可以選擇建立新的 IP 位址，但如果虛擬機器不會有公用 IP 位址，您也可以選取 [(無)]。</span><span class="sxs-lookup"><span data-stu-id="40c56-142">You can choose to create a new IP address or, if your virtual machine will not have a public IP address, you can select **(None)**.</span></span>

   * <span data-ttu-id="40c56-143">**網路安全性群組**︰為虛擬機器指定選用的網路防火牆。</span><span class="sxs-lookup"><span data-stu-id="40c56-143">**Network security group**: Specifies an optional networking firewall for your virtual machine.</span></span> <span data-ttu-id="40c56-144">您可以選取現有防火牆，但如果虛擬機器不會使用網路防火牆，您也可以選取 [(無)]。</span><span class="sxs-lookup"><span data-stu-id="40c56-144">You can select an existing firewall or, if your virtual machine will not use a network firewall, you can select **(None)**.</span></span>

   * <span data-ttu-id="40c56-145">**可用性設定組**︰指定虛擬機器可以加入的選用可用性設定組。</span><span class="sxs-lookup"><span data-stu-id="40c56-145">**Availability set**: Specifies an optional availability set that your virtual machine can belong to.</span></span> <span data-ttu-id="40c56-146">您可以選取現有的可用性設定組、建立新的可用性設定組，如果虛擬機器將不屬於任何可用性設定組，則可選取 [(無)]。</span><span class="sxs-lookup"><span data-stu-id="40c56-146">You can select an existing availability set or create a new availability set or, if your virtual machine will not belong to an availability set, you can select **(None)**.</span></span>

   ![相關聯的資源視窗][CR07]

10. <span data-ttu-id="40c56-148">按一下 [完成] 。</span><span class="sxs-lookup"><span data-stu-id="40c56-148">Click **Finish**.</span></span>  

    <span data-ttu-id="40c56-149">Azure Explorer 工具視窗中便會顯示新的虛擬機器。</span><span class="sxs-lookup"><span data-stu-id="40c56-149">Your new virtual machine is displayed in the Azure Explorer tool window.</span></span>

    ![新虛擬機器][CR08]
