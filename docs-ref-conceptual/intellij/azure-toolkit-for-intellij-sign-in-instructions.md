## <a name="sign-in-to-your-azure-account-automatically"></a><span data-ttu-id="70a13-108">自動登入您的 Azure 帳戶</span><span class="sxs-lookup"><span data-stu-id="70a13-108">Sign in to your Azure account automatically</span></span>

<span data-ttu-id="70a13-109">本節會逐步引導您建立認證檔案，並在檔案中包含您的服務主體資料。</span><span class="sxs-lookup"><span data-stu-id="70a13-109">This section walks you through creating a credentials file that contains your service principal data.</span></span> <span data-ttu-id="70a13-110">在完成此程序後，Eclipse 會使用認證檔案，讓您在每次開啟專案時自動登入 Azure。</span><span class="sxs-lookup"><span data-stu-id="70a13-110">After you have completed this process, Eclipse uses the credentials file to automatically sign you in to Azure each time you open your project.</span></span>

1. <span data-ttu-id="70a13-118">在 [建立驗證檔案] 視窗中，選取您想要使用的訂用帳戶，選擇您的目的地目錄，然後按一下 [啟動]。</span><span class="sxs-lookup"><span data-stu-id="70a13-118">In the **Create Authentication Files** window, select the subscriptions that you want to use, choose your destination directory, and then click **Start**.</span></span>

1. <span data-ttu-id="70a13-120">在 [服務主體建立狀態] 對話方塊中，於系統成功建立檔案之後按一下 [確定]。</span><span class="sxs-lookup"><span data-stu-id="70a13-120">In the **Service Principal Creation Status** dialog box, after your files have been created successfully, click **OK**.</span></span>

   ![[服務主體建立狀態] 對話方塊][A05]

1. <span data-ttu-id="70a13-122">在 [Azure 登入] 視窗中，按一下 [登入]。</span><span class="sxs-lookup"><span data-stu-id="70a13-122">In the **Azure Sign In** window, click **Sign in**.</span></span>

   ![[Azure 登入] 對話方塊][A06]

1. <span data-ttu-id="70a13-124">在 [選取訂用帳戶] 對話方塊中，選取您要使用的訂用帳戶，然後按一下 [確定]。</span><span class="sxs-lookup"><span data-stu-id="70a13-124">In the **Select Subscriptions** dialog box, select the subscriptions that you want to use, and then click **OK**.</span></span>

   ![[選取訂用帳戶] 對話方塊][A07]
