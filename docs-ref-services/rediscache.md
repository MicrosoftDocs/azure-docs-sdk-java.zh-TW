## <a name="example"></a><span data-ttu-id="ecd7d-111">範例</span><span class="sxs-lookup"><span data-stu-id="ecd7d-111">Example</span></span>

```java
    shardInfo.setPassword("<key>"); /* Use your access key. */

    jedis.set("foo", "bar");
```
